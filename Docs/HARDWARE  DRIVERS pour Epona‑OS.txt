# 🟦 Niveau 1 — MMIO  
### 🎯 Objectif  
Lire/écrire dans les registres matériels.

### 🧪 Exemple Forth — Lire un registre MMIO
```
hex
E0000000 mmio-read .
```

### 🧪 Exemple Forth — Écrire un registre MMIO
```
hex
E0000000 00FF00 mmio-write
```

### 🧪 Exemple Forth — Lire un registre APIC
```
hex
FEE00000 20 mmio-read .
```

---

# 🟩 Niveau 2 — PCI  
### 🎯 Objectif  
Scanner le bus PCI, lire Vendor ID, BAR, capabilities.

### 🧪 Exemple Forth — Scanner PCI
```
pci
```

### 🧪 Exemple Forth — Lire Vendor ID
```
0 1 0 0 pci-read .
```

### 🧪 Exemple Forth — Lire BAR0
```
0 1 0 16 pci-read .
```

### 🧪 Exemple Forth — Masquer les flags d’un BAR
```
bar0 FFFFFFF0 and .
```

### 🧪 Exemple Forth — Lister les capabilities
```
: cap-list
    0 1 0 52 pci-read
    begin
        dup 0= if drop exit then
        dup .                     \ offset
        0 1 0 swap pci-read .     \ ID
        0 1 0 swap 1+ pci-read    \ next
    again
;
cap-list
```

---

# 🟧 Niveau 3 — ACPI  
### 🎯 Objectif  
Lire MADT, HPET, Local APIC.

### 🧪 Exemple Forth — Afficher les tables ACPI
```
acpi
```

### 🧪 Exemple Forth — Trouver MADT
```
"APIC" acpi-find .
```

### 🧪 Exemple Forth — Lire l’adresse du Local APIC
```
madT-local-apic-base @ .
```

### 🧪 Exemple Forth — Trouver HPET
```
"HPET" acpi-find .
hpet-base @ .
```

# 🟥 Niveau 4 — APIC  
### 🎯 Objectif  
Activer APIC, configurer le timer.

### 🧪 Exemple Forth — Activer APIC
```
hex
FEE00000 F0 mmio-read
100 or
FEE00000 F0 mmio-write
```

### 🧪 Exemple Forth — Configurer le diviseur du timer
```
hex
FEE00000 3E0 3 mmio-write
```

### 🧪 Exemple Forth — Charger une valeur initiale
```
hex
FEE00000 380 100000 mmio-write
```

### 🧪 Exemple Forth — Mode périodique
```
hex
FEE00000 320 200 mmio-write
```

# 🟦 Niveau 5 — I/O‑APIC  
### 🎯 Objectif

### 🧪 Exemple Forth — Lire le nombre d’entrées IRQ

```

### 🧪 Exemple Forth — Remapper IRQ1 → vecteur 0x21

```
