# Contribution matériel — Epona OS

Pour aider à améliorer la compatibilité matérielle d’Epona :

1. Démarrez Epona sur votre machine.
2. Dans le shell, tapez : `pci save`
3. Récupérez le fichier `PCI.TXT` sur votre clé USB.
4. Allez sur le dépôt GitHub Epona.
5. Ouvrez une nouvelle *Issue*.
6. Donnez un titre du type : `[HW] Dump PCI — <marque modèle>`
7. Attachez votre fichier `PCI.TXT` dans l’Issue.
8. Ajoutez quelques infos sur la machine (CPU, GPU, modèle).
